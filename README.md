# SHIFT_test_task

Тестовое задание для курса от ШИФТ

# Инструкция запуска

## Пакеты для запуска

Используемая версия `Java` и `Javac` - 17.0.8
Также нужен `GNU Make` версии 4.3

## Запуск может происходить самостоятельно из терминала используя:

1. `javac *.java`
2. `java Main -o <директория для файлов-результатов> -s/-f -a -p <префикс для файлов-результатов> <входные файлы через пробел>`

## Или же можно запустить через `Makefile` используя:

1. `make run ARGS="-o <директория для файлов-результатов> -s/-f -a -p <префикс для файлов-результатов> <входные файлы через пробел>"`
2. `make run-short` - пример с короткой статистикой
3. `make run-full` - пример с полной статистикой
4. `make help` - все команды с `make`
